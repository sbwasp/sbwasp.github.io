See OWASP Vulnerable Web Applications Directory Project for the OWASP Vulnerable Web Applications Directory Project and the OWASP Juice Shop Project. The Juice Shop illustrates extending VM-based challenges to using the cloud (Heroku, EC2), containers, Vagrant, and packed distribution of challenges.
