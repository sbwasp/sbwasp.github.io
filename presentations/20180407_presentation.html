11.6.2. Study this week¶

11.6.2.1. nmap nuances¶

The definitive guide to nmap is Chapter 15. Nmap Reference Guide. Would-be pentesters should first learn Port Scanning Basics and Port Scanning Techniques. More advanced users should consult Timing and Performance, Firewall/IDS Evasion and Spoofing, http-waf-detect, and Output.

Not covered above is using xsltproc to process nmap XML output into report-ready HTML:

sudo nmap -sn -PE -oA nmap_sn 192.168.1.0/24
sudo chown $USER.$USER nmap_sn.*
grep Up nmap_sn.gnmap | cut -d" " -f2 > targets.txt
xsltproc nmap_sn.xml -o nmap_sn.html
