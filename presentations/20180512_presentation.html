11.4.2.1.2. asciinema¶

asciinema is a tool to record and play terminal sessions. See asciinema Docs.

pip install -U asciinema

# record a session with at most 2.5 seconds idle time
asciinema rec -i 2.5 curl.cast
curl --tlsv1.3 --head https://sbwasp.github.io/
curl --tlsv1.2 --head https://sbwasp.github.io/
exit
asciinema play curl.cast

11.4.2.2.2. Kubernetes¶

See Kubernetes tutorials starting with Kubernetes Basics. The tutorial uses KataKoda. Also see Kubernetes Design and Architecture.

See Running Kubernetes Locally via Minikube: “Minikube is a tool that makes it easy to run Kubernetes locally. Minikube runs a single-node Kubernetes cluster inside a VM on your laptop for users looking to try out Kubernetes or develop with it day-to-day.”

etcd is the primary storage mechanism for K8s:

etcd is a distributed reliable key-value store for the most critical data of a distributed system, with a focus on being:

  • Simple: well-defined, user-facing API (gRPC)
  • Secure: automatic TLS with optional client cert authentication
  • Fast: benchmarked 10,000 writes/sec
  • Reliable: properly distributed using Raft

etcd is written in Go and uses the Raft consensus algorithm to manage a highly-available replicated log.

For a more complex introduction that requires a Google Cloud Platform account, see Google Kubernetes Engine codelab.

For AWS see Getting Started with Amazon Elastic Container Service and the Tutorial: Creating a Cluster with a Fargate Task Using the ECS CLI or Getting Started with Amazon ECS using Fargate.
