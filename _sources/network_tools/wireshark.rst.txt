*********
Wireshark
*********

Consult the `Wireshark User's Guide <http://www.wireshark.org/docs/wsug_html_chunked/>`_ and `Wireshark Wiki`_ for information on using Wireshark. Especially useful are:

* `Wi-Fi (WLAN, IEEE 802.11) <http://wiki.wireshark.org/Wi-Fi>`_
* `WLAN (IEEE 802.11) capture setup <http://wiki.wireshark.org/CaptureSetup/WLAN>`_
* `802.11 frames graphics <https://www.google.com/search?q=802.11+frames+graphic&tbm=isch>`_
* `Display Filter Reference: IEEE 802.11 wireless LAN <http://www.wireshark.org/docs/dfref/w/wlan.html>`_
* `Wireshark Online Tools <http://www.wireshark.org/tools/>`_ including:

  * `String-Matching Capture Filter Generator <http://www.wireshark.org/tools/string-cf.html>`_
  * `WPA PSK (Raw Key) Generator <http://www.wireshark.org/tools/wpa-psk.html>`_
  * `OUI Lookup Tool <http://www.wireshark.org/tools/oui-lookup.html>`_
